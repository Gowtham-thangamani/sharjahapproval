Setup Required - Connect Google Analytics

To show real analytics data, connect your GA4 property with a service account:

  1. Go to Google Cloud Console
  2. Create a project and enable Google Analytics Data API
  3. Go to IAM & Admin > Service Accounts > Create Service Account
  4. Download the JSON key file
  5. Upload it to your server as admin/data/ga-credentials.json
  6. In GA4 Admin > Property Access Management, add the service account email as Viewer
